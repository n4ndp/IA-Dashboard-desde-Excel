# Prueba Técnica: Desarrollo Web Full-Stack

## 1. Objetivo

Construir una aplicación web funcional (Frontend + Backend) utilizando herramientas de aceleración de IA, demostrando habilidades en arquitectura de datos, integración de APIs de Inteligencia Artificial y despliegue continuo.

## 2. El Reto (Elige un Caso de Uso)

El candidato debe seleccionar una de las siguientes propuestas para desarrollar:

- **A. IA Dashboard desde Excel:** Una aplicación que reciba el archivo Excel adjunto, procese la información en el backend y genere un dashboard interactivo con gráficos y un resumen ejecutivo generado por IA sobre los datos.
- **B. Generador de CV de Alto Impacto:** Un formulario dinámico donde el usuario ingresa sus intereses y experiencia. La IA debe procesar esta información para redactar un perfil profesional optimizado y generar un CV descargable.
- **C. Directorio de Proveedores Inteligente (CRUD):** Una plataforma para gestionar proveedores de servicios (Crear, Leer, Actualizar, Eliminar) que incluya un buscador inteligente o un sistema de categorización automática por IA basado en la descripción del proveedor.

## 3. Requisitos Técnicos

### Stack Tecnológico

- **Frontend:** Angular o VueJS.
- **Backend:** FastAPI (Python) o NestJS (Node.js).
- **Base de Datos:** Supabase, Firestore, MySQL, BigQuery o PostgreSQL.
- **Entorno de Desarrollo (Sugerido):** Se permite y recomienda el uso de herramientas de vibecoding como Lovable, Bolt.new, Firebase Studio, Cursor o Replit. También puede usarse un entorno cloud (GCP/AWS).

### Criterios de Evaluación

1. **Funcionalidad Full-Stack:** Conexión exitosa entre el frontend y el backend.
2. **Arquitectura de Datos:** Explicación clara del modelo de datos y por qué se eligió esa estructura (ej. relacional vs no-relacional).
3. **Integración de IA (Puntos Extra):** Implementación de servicios de Gemini API u OpenAI para tareas como:
   - Análisis de sentimientos.
   - Resumen de documentos/datos.
   - Extracción de palabras clave.
4. **Despliegue y DevOps (Puntos Extra):** Integración con GitHub para despliegue automático (ej. Vercel, Netlify, Railway, o GitHub Actions).

## 4. Entregables

1. **URL del Repositorio:** Código fuente organizado en GitHub/GitLab.
2. **URL de la App en vivo:** Enlace a la aplicación desplegada (si aplica).
3. **Documentación Breve (README):**
   - Diagrama o explicación de la arquitectura.
   - Instrucciones para ejecutar en local.
   - Descripción de cómo se utilizó la IA en el proceso de desarrollo y dentro de la app.
